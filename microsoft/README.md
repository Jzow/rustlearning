# microsoft

此模块是按照microsoft所推出的教程进行各个章节的源代码学习，详情参考网站：[microsoft-learn-rust](https://docs.microsoft.com/zh-cn/learn/paths/rust-first-steps/) 

### 章节如下：

* 创建 Rust 程序
* 使用 Rust 中的 if/else 表达式测试条件
* 使用循环来循环访问 Rust 中的数据
* 在 Rust 中处理错误
* 了解 Rust 如何管理内存
* 实现泛型类型和特征
* 探索模块、包和第三方箱
* 写入自动测试
* 构建命令行待办事项列表程序
